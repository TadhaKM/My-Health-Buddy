# Future You Demo Script (Under 60 Seconds Each)

## Narrative 1 - Healthy Baseline (40-50s)
1. Click preset `Healthy`.
2. Call out the timeline progression at 0, 5, 10, and 20 years.
3. Explain that all organs remain in low-strain bands and move gradually.
4. Highlight the value: users can see stable outcomes when core habits are strong.

Key line:
"With balanced habits, the projection stays low and flat, so users immediately understand what protective behavior looks like."

## Narrative 2 - Smoker Profile (45-60s)
1. Click preset `Smoker`.
2. Move timeline to 20 years.
3. Point out lungs and heart rising faster than healthy baseline.
4. Show AI summary and organ insight card aligning with visible trend.

Key line:
"The model makes the lungs-heart impact obvious by year 5 and unmistakable by year 20, which is exactly what judges need to see in seconds."

## Narrative 3 - Stress Combo (45-60s)
1. Click preset `Stress Combo`.
2. Move timeline across 5 -> 10 -> 20 years.
3. Emphasize liver-brain-body fat acceleration from combined alcohol/sleep/diet/exercise pressure.
4. Type a short chat message (for example, "I barely sleep and drink most nights") and show updated habits + simulation refresh.

Key line:
"This is not a diagnosis tool - it is an educational mirror that helps users connect daily routines to long-term organ strain."

## Presenter Tips
- Keep one hand on the timeline selector; visual progression is the fastest proof.
- Always compare against `Healthy` once so contrast is immediate.
- If AI response is slow, fallback still preserves believable deterministic output.
